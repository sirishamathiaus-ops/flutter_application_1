# flutter_application_1

**A new Flutter project.**

A simple Notes app built with Flutter — create, read, update, and delete ,search notes easily.

**📌 About**
This project is a minimalist notes application made with Flutter. It helps you organize your thoughts by creating, editing, searching and deleting notes quickly and efficiently. Designed with simplicity in mind.

**📝 Features**

Add Notes – Create new notes with a title and description.

Edit Notes – Update your existing notes easily.

Delete Notes – Remove notes you no longer need.

Search Notes – Quickly find notes by keyword.

User-Friendly UI – Clean and simple interface for easy navigation.

**💻 Tech Stack**

Flutter – For building cross-platform mobile apps.

Dart – Programming language used with Flutter.

Local Storage  – To store notes on the device.

**🚀 Installation**

1.Clone the repository

git clone https://github.com/sirishamathiaus-ops/flutter_application_1

2.Navigate to project folder

cd flutter_application_1

3.Install dependencies

flutter pub get

4.Run the app

flutter run/flutter run -d chrome  

**🛠️ Usage**

1.Open the app.

2.Tap the Add Note button to create a new note.

3.Edit or delete notes by tapping the note.

4.Use the Search bar to quickly find notes.
